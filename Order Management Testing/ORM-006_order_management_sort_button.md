# ORM-006: Order Management – Sort Button Functionality

> **Summary:** Verify that the **Sort** button sorts orders correctly based on selected criteria.

**Preconditions:**  
- Vendor is logged in.  
- Multiple orders exist.

| # | Step | Expected Behavior |
|---|------|-------------------|
| 1 | Click the **Sort** button. | Dropdown menu appears with sorting options (e.g., Date, Total, Status). |
| 2 | Select a sorting option. | System rearranges orders accordingly. |

**Actual Result:**  
Sort button fails to respond or does not rearrange orders.  

**Status:**  
Fail ❌
