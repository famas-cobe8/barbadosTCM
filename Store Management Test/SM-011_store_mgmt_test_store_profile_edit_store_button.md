# SM-011: Store Management Testing – Edit Store Profile Button

> **Summary:** Verify that the vendor can successfully initiate store profile editing by clicking the **Edit** (pencil) icon.

**Preconditions:**  
- Vendor is logged in.  
- Store profile section is visible.  
- Store is active.

### Scenario 1: Initiate store profile edit

| # | Step | Expected Behavior |
|---|------|-------------------|
| 1 | Navigate to the **Store Profile** section. | Store details are displayed with an **Edit** icon. |
| 2 | Click the **Edit** icon. | System opens the store profile editing modal. |
| 3 | Verify modal contents. | Editable fields for store name, description, and operating hours are visible. |

**Actual Result:**  
System successfully opens the store profile editing modal with editable fields.  

**Status:**  
Pass ✅  

**Post‑conditions:**  
- Store profile modal is active and ready for editing.
