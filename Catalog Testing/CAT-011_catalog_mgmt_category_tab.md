# CAT-011: Catalog Management – Category Tabs

> **Summary:** Verify that each category tab filters products correctly.

**Preconditions:**  
- Vendor is logged in.  
- Catalog page is accessible.  
- Products exist under multiple categories.

| # | Step | Expected Behavior |
|---|------|-------------------|
| 1 | Click a category tab (e.g., **Beverages**, **Breakfast**, **Illegal**). | System displays only products belonging to the selected category. |
| 2 | Switch between categories. | Product list updates accordingly without delay. |
| 3 | Verify active tab indicator. | Selected tab is highlighted with an underline. |

**Actual Result:**  
Category tabs filter and display products correctly.  

**Status:**  
Pass ✅
