# MSG-V-004: Messaging Module – Message Timestamp Display

> **Summary:** Verify that each message displays the correct timestamp.

**Preconditions:**  
- Vendor is logged in.  
- User is in the **Messages menu page**.  
- Chat window contains multiple messages.

| # | Step | Expected Behavior |
|---|------|-------------------|
| 1 | Observe timestamps beside messages. | Each message shows accurate time of sending or receiving. |
| 2 | Compare with system time. | Timestamps match the actual time recorded. |

**Actual Result:**  
Timestamps displayed accurately for all messages.  

**Status:**  
Pass ✅
