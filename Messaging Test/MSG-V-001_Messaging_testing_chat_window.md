# MSG-V-001: Messaging Module – Open Chat Window

> **Summary:** Verify that the vendor can open a chat window with a customer.

**Preconditions:**  
- Vendor is logged in.  
- User is in the **Messages menu page**.  
- At least one conversation exists.

| # | Step | Expected Behavior |
|---|------|-------------------|
| 1 | Navigate to the **Messages** menu. | Message list loads successfully. |
| 2 | Click on a customer conversation. | Chat window opens displaying past messages. |

**Actual Result:**  
Chat window opens and displays conversation history correctly.  

**Status:**  
Pass ✅
