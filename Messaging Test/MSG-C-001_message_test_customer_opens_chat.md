# MSG-C-001: Customer opens chat interface

> **Category:** Messaging  
> **Summary:** Verify that the customer can open the chat interface with a vendor successfully.

**Preconditions:**  
- Customer is logged in.  
- Vendor is active and available for chat.

### Scenario 1

| # | Step | Expected Behavior |
|---|------|-------------------|
| 1 | Customer navigates to the vendor’s shop page. | Chat icon or button is visible. |
| 2 | Customer clicks on **Chat with Store**. | Chat interface opens showing vendor name and status (e.g., “Active now”). |
| 3 | Verify chat input field and send button visibility. | Input field and send button are displayed correctly. |
| 4 |  |  |
| 5 |  |  |
| 6 |  |  |

**Actual Result:**  
Chat interface opens successfully with vendor details and active status displayed.  

**Status:**  
Pass  

**Post‑conditions:**  
- Customer can begin sending messages.
