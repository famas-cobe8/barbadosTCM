# MSG-V-005: Messaging Module – Customer Status Indicator

> **Summary:** Verify that the customer’s online status is displayed correctly.

**Preconditions:**  
- Vendor is logged in.  
- User is in the **Messages menu page**.  
- Chat window with a customer is open.

| # | Step | Expected Behavior |
|---|------|-------------------|
| 1 | Observe the customer header section. | Customer name and profile icon are visible. |
| 2 | Check status indicator. | “Active now” appears when customer is online; otherwise shows last active time. |

**Actual Result:**  
Customer status displayed correctly.  

**Status:**  
Pass ✅
