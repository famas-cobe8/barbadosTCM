# MSG-V-003: Messaging Module – Receive Message

> **Summary:** Verify that the vendor can receive messages from a customer.

**Preconditions:**  
- Vendor is logged in.  
- User is in the **Messages menu page**.  
- Customer sends a message.

| # | Step | Expected Behavior |
|---|------|-------------------|
| 1 | Wait for incoming message. | New message appears in chat window with timestamp. |
| 2 | Verify sender label. | Message shows correct sender identity (Customer). |

**Actual Result:**  
Incoming message displayed correctly with sender and timestamp.  

**Status:**  
Pass ✅
