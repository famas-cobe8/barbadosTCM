# MSG-V-002: Messaging Module – Send Message

> **Summary:** Verify that the vendor can send a message to a customer.

**Preconditions:**  
- Vendor is logged in.  
- User is in the **Messages menu page**.  
- Chat window with a customer is open.

| # | Step | Expected Behavior |
|---|------|-------------------|
| 1 | Type a message in the input field. | Text input is accepted. |
| 2 | Click the **Send** button (paper plane icon). | Message is delivered and appears in the chat window with timestamp. |

**Actual Result:**  
Message sent successfully and displayed in chat history.  

**Status:**  
Pass ✅
