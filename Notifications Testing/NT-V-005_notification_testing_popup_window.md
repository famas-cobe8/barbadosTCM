# NT-V-005: Notification Test – Popup Notification Window

> **Summary:** Verify that clicking the **Alert (bell)** icon opens the notification popup window correctly when vendor is logged in.

**Preconditions:**  
- Vendor is logged in.  
- Notifications exist for new orders, cancelled orders, new messages from customers, and completed orders.

### Scenario 1: Open notification popup

| # | Step | Expected Behavior |
|---|------|-------------------|
| 1 | Click the **Alert (bell)** icon. | Notification popup window appears immediately. |
| 2 | Observe the content. | Notifications are displayed with correct formatting, icons, and timestamps. |
| 3 | Close the popup. | Popup closes smoothly without errors. |

**Actual Result:**  
System successfully opens and displays the notification popup window upon clicking the alert icon.  

**Status:**  
Pass ✅  

**Post‑conditions:**  
- Notification popup is accessible and functional.
