# NAV-0001: Navigation Tabs - Shops Tab Redirection

> **Category:** Navigation Tabs  
> **Summary:** Verify that clicking the Shops navigation tab successfully routes the user to the shop directory list.

**Preconditions:**  
- Customer is logged into the web application.
- Navigation tabs are visible.

### Scenario 1

| # | Step | Expected Behavior |
|---|------|-------------------|
| 1 | Click on the **Shops** navigation tab. | The page body switches to the shop directory list view. |

**Actual Result:**  
The application successfully updates the view and routes the user to the shop directory list page.  

**Status:**  
Pass  

**Post-conditions:**  
- The user is left viewing the active shops directory listings.
