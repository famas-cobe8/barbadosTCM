# NAV-003: Navigation Tabs - Delivery Tab Redirection

> **Category:** Navigation Tabs  
> **Summary:** Verify that clicking the Delivery navigation tab successfully routes the user to the order tracking panel.

**Preconditions:**  
- Customer is logged into the web application.
- Navigation tabs are visible.

### Scenario 1

| # | Step | Expected Behavior |
|---|------|-------------------|
| 1 | Click on the **Delivery** navigation tab. | The page body switches to the active order tracking panel. |

**Actual Result:**  
The application successfully updates the view and routes the user to the delivery tracking view.  

**Status:**  
Pass  

**Post-conditions:**  
- The user is left viewing their order tracking information history.
