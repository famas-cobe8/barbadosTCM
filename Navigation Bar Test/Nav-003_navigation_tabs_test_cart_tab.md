# NAV-004: Navigation Tabs - Cart Tab Redirection

> **Category:** Navigation Tabs  
> **Summary:** Verify that clicking the Cart navigation tab successfully routes the user to the cart dashboard.

**Preconditions:**  
- Customer is logged into the web application.
- Navigation tabs are visible.

### Scenario 1

| # | Step | Expected Behavior |
|---|------|-------------------|
| 1 | Click on the **Cart** navigation tab. | The page body switches to the primary cart dashboard overview. |

**Actual Result:**  
The application successfully updates the view and routes the user to the multi-cart dashboard section.  

**Status:**  
Pass  

**Post-conditions:**  
- The user is left viewing their item checkout baskets.
