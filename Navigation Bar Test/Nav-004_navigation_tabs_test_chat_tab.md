
# NAV-004: Navigation Tabs - Chats Tab Redirection

> **Category:** Navigation Tabs  
> **Summary:** Verify that clicking the Chats navigation tab successfully routes the user to the message center.

**Preconditions:**  
- Customer is logged into the web application.
- Navigation tabs are visible.

### Scenario 1

| # | Step | Expected Behavior |
|---|------|-------------------|
| 1 | Click on the **Chats** navigation tab. | The page body switches to the direct message center panel. |

**Actual Result:**  
The application successfully updates the view and routes the user to the chat conversations view.  

**Status:**  
Pass  

**Post-conditions:**  
- The user is left viewing active direct message histories with merchants.
